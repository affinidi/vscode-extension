# VSCode Extension for Affinidi

## Prerequisites

1. [AWS Serverless Application Model
](https://aws.amazon.com/serverless/sam/):
```
brew tap aws/tap
brew install aws-sam-cli
```
2. Docker running.
3. `npm install`
4. `npm run compile`
